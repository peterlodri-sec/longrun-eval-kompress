# About This Project

This exists for two reasons: **experimentation** and **knowledge sharing**.

Nothing else.

No institutional backing. No grant money. No product roadmap. Just someone
with a question ("can a small model learn which tokens matter?"), a budget
($38.95), and the belief that research should be open, honest, and cheap
enough that anyone can replicate it.

If you're here because you want to learn, fork it. If you're here because
you want to build, use it. If you're here because you're curious, that's
enough.

Every dead end is documented. Every number is sourced. The 11 models that
didn't work are published alongside the ones that did. That's the point.

---

Built by Peter Lodri. Funded by the community. Powered by curiosity.
